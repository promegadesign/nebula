Tiles

Published: September 1, 2021   |   Updated: September 27, 2021

Page Status#

This page is currently under construction. Use our Contact Form if you have questions or feedback.

Overview#

Tiles are large, clickable elements. They are usually used in a group of related items, such as child items under a parent item or resources for the same product/topic. They are distinct from Cards in that they only link to one target instead of multiple targets.

Tile Types#

Resource Tiles#

  • Tiles are built automatically using content stored on the linked item
    • Icon, label, heading, and background image can be changed by editing the linked item, but edits will be displayed everywhere the linked item is shown in a tile
  • No control over styling (CSS, colors, etc.)
  • Primarily used for sharing related resources to a central topic or theme
  • Example: Cell Biology Resources

Descendants Grid Tiles (v1)#

  • Built automatically based on children of parent item
  • Content shown on tile is pulled from child item (heading and description)
  • One color for background
  • No control over styling
  • Example: Cell Biology Product Categories

Descendants Grid Tiles (v2)#

  • Use on the home page and promotions landing page
  • Example: Promotional Offers

Custom-Built Tiles#

  • Tiles can be built using the grid parent component and tile child components
  • Select components can be added to the tile, including
    • Heading
    • Image
    • Paragraph text
    • Horizontal rule
    • Background color/image
  • Elements within the tile can be aligned to left, right, or center
  • Text color can be set to dark/black or white

Product Tiles#

  • Often referred to as “Product Cards” as the design is distinct from other tiles on the site
  • Example Product Group Page: Cytokine and Growth Factor Bioassays
  • Example Product Detail Page: Related Products for IL-12 Bioassay

Guidelines#

Use Tiles When#

  • Displaying child items (e.g., sub-pages or sub-categories under a parent category).
  • Displaying related items/content under a specific topic or theme.
  • Drilling down the path to product.
  • Linking to a single destination.
  • Displaying less content, including one of each:
    • Heading
    • Image
    • Icon
    • Paragraph text
    • CTA

Don’t Use Tiles When#

  • Linking to multiple destinations. Instead, use a link or button.
  • Linking to a file download. Instead, use a link , button, or the File Download component.
  • Displaying complex content, e.g., multiple paragraphs, images, headings and subheadings.
